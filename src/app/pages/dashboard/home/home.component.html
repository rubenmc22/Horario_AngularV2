Sistema de Gestión de Horarios

Presentación


Sistema de gestión que permitirá mantener al día la información de asignaturas, cursos y docentes, así como organizar las cargas académicas para cada curso de una manera cómoda y flexible. Adicionalmente, y como función principal, el sistema permite el generado automático de los horarios académicos de forma instantánea.
